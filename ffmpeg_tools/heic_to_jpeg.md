# 🖼️ Convert HEIC to JPEG on macOS

macOS includes a built-in tool (Automator) that can convert HEIC images to JPEG format.

## 📝 Steps to Create a Quick Action

1. Open **Automator** on your Mac.
2. Choose **Quick Action** as the type of document.
3. In the search bar, look for and add **“Change Type of Images”**.
4. Set the input type to: `Image files`.
5. Choose the output format: `JPEG`.
6. Save the Quick Action as `HEICtoJPEG`.

## 📂 How to Use

- In Finder, right-click on any `.HEIC` file.
- Select **Services > HEICtoJPEG** to convert it.
